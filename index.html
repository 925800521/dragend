dragend JS

a touch ready, full responsive, content swipe script

  • dragend JS is a touch ready, full responsive, content swipe script. It's open source and uses hammer.js for observing multi-touch gestures. It also can, but don't have to, be used as a jQuery plugin

    The current version is 0.2.0 release candidate 1

    Features

    • horizontal and vertical swiping
    • mobile and desktop ready
    • lightweight
    • keyboard navigation
    • fullscreen or boxed
    • contend scribe
    • resize adjustment
    • overscroll
    • full responsive

    Getting started

  • Getting started

    To activate dragend JS just call the function on a jQuery element:

    $("#demo").dragend();

  • Configuration

    You can configure dragend.js by overwriting the default config. You can also pass callback function for onSwipeStart and onSwipeEnd.

    If you want to swipe after initializing just call dragend again on the element with the direction you want to swipe to.

    For directly jumping or scrolling to a page call dragend again on the element and add jumpToPage or scrollToPage in the option object.
