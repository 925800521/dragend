dragend JS

a jQuery plugin for content swiping

  • dragend JS is a swipe plugin for jQuery. It's open source and uses hammer.js for observing multi-touch gestures.

    The current version is 0.1.0.

    Features

    • horizontal and vertical swiping
    • lightweight (1.55KB gzipped)
    • keyboard navigation
    • fullscreen or boxed
    • contend scribe
    • resize adjustment
    • overscroll

    Download

    • Download development version (unminified, with comments)
    • Download production version (1.55KB minified & compressed)

    Browser Support

    • Firefox 3.5+
    • Chrome
    • IE 8+
    • Safari 4+
    • Safari on iOS 5+
    Getting started

  • Getting started

    To activate dragend JS just call the function on a jQuery element:

    $("#demo").dragend();

  • Configuration

    You can configure dragend.js by overwriting the default config. You can also pass callback function for onSwipeStart and onSwipeEnd.

    If you want to swipe after initializing just call dragend again on the element with the direction you want to swipe to.

    For directly jumping or scrolling to a page call dragend again on the element and add jumpToPage or scrollToPage in the option object.
