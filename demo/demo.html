dragend JS

a jQuery plugin for content swiping

  • dragend.js is a swipe plugin for jQuery. It's open source and uses hammer.js for observing multi-touch gestures.

    Features

    • Keyboard navigation
    • Fullscreen or boxed
    • Lightweight (853 bytes gzipped)
    • Contend scribe

    Browser Support

    • Firefox 3.5+
    • Chrome
    • IE 8+
    • Safari 4+
    • Safari on iOS 5+
    • Opera 11+
