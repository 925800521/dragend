dragend JS

a jQuery plugin for content swiping

  • dragend JS is a swipe plugin for jQuery. It's open source and uses hammer.js for observing multi-touch gestures.

    The current version is 0.1.0 beta. The first release is near!

    Features

    • horizontal and vertical swiping
    • lightweight (1.08KB gzipped)
    • keyboard navigation
    • fullscreen or boxed
    • contend scribe
    • resize adjustment
    • overscroll

    Browser Support

    • Firefox 3.5+
    • Chrome
    • IE 8+
    • Safari 4+
    • Safari on iOS 5+
    • Opera 11+
    Getting started

  • Getting started

    To activate dragend JS just call the function on a jQuery element:

    $("#swipe-container").dragend();
