dragend JS

a jQuery plugin for content swiping

  • dragend.js is a swipe plugin for jQuery. It's open source and uses hammer.js for observing multi-touch gestures.

    The current version is 0.1.0 alpha. The first release is near!

    Features

    • Horizontal and vertical swiping
    • Lightweight (853 bytes gzipped)
    • Keyboard navigation
    • Fullscreen or boxed
    • Contend scribe
    • Resize adjustment

    Browser Support

    • Firefox 3.5+
    • Chrome
    • IE 8+
    • Safari 4+
    • Safari on iOS 5+
    • Opera 11+
