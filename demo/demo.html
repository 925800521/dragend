dragend JS

a jQuery plugin for content swiping

  • dragend.js is a lightweight (853 bytes gzipped) swipe plugin for jQuery. It's open source and uses hammer.js for tracking multi-touch gestures.

    Features

    • Keyboard navigation

    Browser Support
